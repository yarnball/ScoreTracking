# Score counter using react state
* see it live! https://codesandbox.io/s/l5nlQ75A6

*Using React 15.4.0, React Router 4, and Material UI*

#### Dev:
* ```npm install```
* ```npm start```
* Visit ```localhost:3000```
* Different port? Run ```PORT=3001 npm start```

#### To build/Production:
* `npm run build`
* see "/build" folder
